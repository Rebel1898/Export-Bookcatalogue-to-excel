# Export-Bookcatalogue-to-excel
An easy UI to convert the CSV export from BookCatalogue (https://www.book-catalogue.com/) to an excel file without headaches.
It may also be used for any CSV,but it was designed with that purpose on mind.

The app is designed with electron and releases are available for Linux and Windows.

# Use
The use is straightforward. Open the app, load the CSV obtained from BookCatalogue. A grid with the table will be shown. 
You may add, delete, rename or reorder the columns. When everythins is as desired just export it to xlsx.

# Build & Run

1. Install [Node.js](https://nodejs.org/) (LTS recommended).
2. Clone this repository and enter the project folder:
3. Install dependencies with npm install.
4. Run in developer mode with npm start
